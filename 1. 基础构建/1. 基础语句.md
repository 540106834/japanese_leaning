#  **基础句型整理（阶段 1）**

## 1️ **です／ます（敬体句）**

* 功能：表达“是/在/做……”的礼貌句型

* 构成：

  * **名词 + です** → 表示“是……”
    例：

    * わたしは ジン **です**。
      → 我是 Jin。
    * これは ほん **です**。
      → 这是书。

  * **动词 + ます** → 表示“做……”（敬体）
    例：

    * たべます → 吃
    * いきます → 去
    * みます → 看

* 小技巧：

  * です = 名词/形容词的礼貌结尾
  * ます = 动词的礼貌结尾

---

## 2️ **ある／いる（有 / 在）**

* 功能：表示存在

* 区别：

  | 用法 | 例子        | 说明         |
  | -- | --------- | ---------- |
  | ある | ほんが あります。 | 东西/无生命物品存在 |
  | いる | ねこが います。  | 生物/有人/动物存在 |

* 句型：

  * **X が あります/います** → “有 X / 在 X”
  * **場所 に X が あります/います** → “在某地有 X / 在某地存在 X”
    例：

    * にわに ねこが います。 → 花园里有猫。
    * つくえの 上に ほんが あります。 → 桌子上有书。

---

## 3️ **これ／それ／あれ（指示词）**

* 功能：指示“这/那/那边的”

* 区别：

  | 指示词 | 意思        | 用法示例               |
  | --- | --------- | ------------------ |
  | これ  | 这个（离自己近）  | これは りんごです。 → 这是苹果。 |
  | それ  | 那个（离对方近）  | それは ペンです。 → 那是笔。   |
  | あれ  | 那个（离双方都远） | あれは 山です。 → 那是山。    |

* 衍生用法：

  * どれ → 哪一个？
    例：どれが あなたの ほんですか？ → 哪本是你的书？

---

## 4️ **结合使用示例**

* 例 1：

  ```
  これは ねこです。
  → 这是一只猫。
  ```
* 例 2：

  ```
  にわに ねこが います。
  → 花园里有猫。
  ```
* 例 3：

  ```
  それは ほんですか？
  → 那是书吗？
  ```
* 例 4：

  ```
  あれは おおきい いぬです。
  → 那边是只大狗。
  ```
